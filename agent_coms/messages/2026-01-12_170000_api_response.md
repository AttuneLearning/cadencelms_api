### 2. Role Display Labels
Available via two methods:

**Option A: Shared Contract (Recommended)**
Import `ROLE_DISPLAY` from `contracts/api/lookup-values.contract.ts`:
```typescript
import { ROLE_DISPLAY } from '@contracts/api/lookup-values.contract';

// Usage:
const displayName = ROLE_DISPLAY['instructor']; // "Instructor"
const displayName = ROLE_DISPLAY['department-admin']; // "Department Admin"
const displayName = ROLE_DISPLAY['system-admin']; // "System Admin"
```

**Option B: API Lookup Endpoint**
```
GET /api/v2/lookup-values?category=role
Authorization: Bearer <token>
```
Returns:
```json
[
  { "key": "instructor", "displayAs": "Instructor", ... },
  { "key": "department-admin", "displayAs": "Department Admin", ... },
  { "key": "system-admin", "displayAs": "System Admin", ... },
  ...
]
```
